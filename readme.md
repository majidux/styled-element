# Styled element

### [npm link](https://www.npmjs.com/package/styled-element)

#### How to install

###### With npm:

```javascript
npm i styled-element
```

###### With yarn:

```javascript
yarn add styled-element
```

#### How to use:

```javascript
import { View } from 'styled-element';
const Example = () => {
  return (
    <View backgroundColor='pink'>
      <View tag='span' color='red'>
        Hello
      </View>
    </View>
  );
};
```
